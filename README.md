#  Строительный магазин

Веб-приложение интернет-магазина строительных материалов.
Состоит из backend части на Spring Boot и frontend на React.

Приложение позволяет просматривать товары, управлять каталогом, оформлять заказы и взаимодействовать с системой как пользователь или администратор.

---

## Запуск backend (Spring Boot)

```bash
cd backend
mvn spring-boot:run
```

## Запуск frontend (React)

```bash
cd frontend
npm i
npm start
```
